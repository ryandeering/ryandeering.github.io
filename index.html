DEASP Project

This was a project I created out of my own volition in response to organisation stress during the COVID-19 pandemic. I created various Grafana dashboards to monitor and log web performance variables such as time to first byte, document complete time and identifiying potential factors to improve web performance on crucial websites related to the COVID-19 pandemic.

I used a combination of JavaScript for the WebPageTest API, InfluxDB for the time-series database and Grafana to graph the data logged in InfluxDB. This project was responded to well within the DEASP. I containerized the project using Docker and wrote documentation for after I left my internship.
